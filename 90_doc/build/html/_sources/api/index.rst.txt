Référence API
=============

Documentation API Python générée automatiquement depuis les docstrings (format Google).

**Exemples pratiques** : chaque module contient des exemples code complets.

**Guides utilisateur** : voir :doc:`../usage` pour contexte et workflows.

-------------------

Organisation
------------

* **utils** : Fonctions utilitaires (couleurs, thème graphique)
* **visualization** : Modules d'analyse et génération de graphiques
* **data** : Chargement et mise en cache des données
* **infrastructure** : Logging, base de données, utilitaires EDA

Tous les modules suivent les conventions :

* Docstrings Google style
* Type hints Python
* PEP8 compliance
* Coverage >= 90%
